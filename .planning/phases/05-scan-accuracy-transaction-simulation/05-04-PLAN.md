# Plan 05-04: Overlay Balance Delta Display

## Objective

Surface the simulation results in the overlay before the Claude streaming explanation. The user sees "+0.5 SOL  -1000 USDC" in a compact delta row immediately when the overlay opens — deterministic, not AI-generated. The simulate call is fire-and-forget with a 2s timeout; if it fails the overlay shows nothing in that slot and Claude's explanation streams normally.

## Context

- `overlay.ts` exports: `showOverlay`, `hideOverlay`, `updateRow`, `appendStream`, `onDecision`, `setVerdict`
- Add new export: `updateSimulation(deltas: SimDelta[], solChangeLamports: number): void`
- The delta row sits between the check rows and the Claude stream text — insert it as a new DOM element in the overlay card
- Content.ts calls `/api/simulate` via `fetch` with `AbortController` (2s timeout), then calls `updateSimulation()` with result
- `VITE_API_BASE` is already used for the worker URL in content.ts

## Tasks

<task id="1" name="Add updateSimulation export to overlay.ts">

In `apps/extension/src/overlay.ts`:

1. Add `SimDelta` interface near the top (mirrors the worker type):
   ```ts
   export interface SimDelta {
     mint: string
     change: number
     decimals: number
     uiChange: string
   }
   ```

2. Add a `simRowRef` module-level variable (alongside other element refs):
   ```ts
   let simRowRef: HTMLElement | null = null
   ```

3. In `buildCard()` (where the card DOM is assembled), after the check-rows container and before the stream text area, create the sim row element:
   ```ts
   const simRow = document.createElement('div')
   simRow.className = 'walour-sim-row'
   simRow.style.cssText = 'display:none; padding:6px 12px; font-size:12px; color:var(--text-muted,#8B949E); letter-spacing:0.02em; font-family:inherit;'
   card.appendChild(simRow)
   simRowRef = simRow
   ```

4. Add `updateSimulation` export:
   ```ts
   export function updateSimulation(deltas: SimDelta[], solChangeLamports: number): void {
     if (!simRowRef) return
     const parts: string[] = []
     if (solChangeLamports !== 0) {
       const sol = solChangeLamports / 1_000_000_000
       parts.push((sol >= 0 ? '+' : '') + sol.toFixed(4) + ' SOL')
     }
     for (const d of deltas) {
       parts.push(d.uiChange + ' ' + d.mint.slice(0, 4) + '...')
     }
     if (parts.length === 0) return
     simRowRef.textContent = parts.join('   ')  // en-space separator
     simRowRef.style.display = 'block'
     // Color: any negative = warn color, all positive = safe color
     const hasLoss = solChangeLamports < 0 || deltas.some(d => d.change < 0)
     simRowRef.style.color = hasLoss ? 'var(--danger,#EF4444)' : 'var(--safe,#22C55E)'
   }
   ```

5. In `hideOverlay()`, reset simRowRef:
   ```ts
   simRowRef = null
   ```

<task id="2" name="Call /api/simulate from content.ts">

In `apps/extension/src/content.ts`:

1. Import `updateSimulation` and `SimDelta` from `./overlay`

2. After `showOverlay()` is called (in the wallet intercept handler), add a fire-and-forget simulate call:
   ```ts
   // Fire-and-forget simulation — 2s timeout, never blocks overlay
   ;(async () => {
     try {
       const controller = new AbortController()
       const timeoutId = setTimeout(() => controller.abort(), 2_000)
       const apiBase = (import.meta.env.VITE_API_BASE as string) ?? 'http://localhost:3001'
       const res = await fetch(`${apiBase}/api/simulate`, {
         method: 'POST',
         headers: { 'Content-Type': 'application/json' },
         body: JSON.stringify({ txBase64 }),
         signal: controller.signal,
       })
       clearTimeout(timeoutId)
       if (res.ok) {
         const data = await res.json() as { success: boolean; solChangeLamports: number; deltas: SimDelta[] }
         if (data.success && (data.deltas.length > 0 || data.solChangeLamports !== 0)) {
           updateSimulation(data.deltas, data.solChangeLamports)
         }
       }
     } catch {
       // Timeout or network error — skip silently
     }
   })()
   ```

   Note: `txBase64` is the base64-encoded transaction already available in the intercept handler scope (it's used to send to `/api/scan`). If it's not already a variable, extract it: `const txBase64 = Buffer.from(tx.serialize()).toString('base64')`
